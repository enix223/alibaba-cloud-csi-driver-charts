# Alibaba cloud csi driver helm chart

This repo keep tracking of kubernetes-sigs/alibaba-cloud-csi-driver, and push charts to enix223/alibaba-cloud-csi-driver if new chart released.

## How to use this chart

```sh
helm install alibaba-cloud-csi-driver \
    oci://ghcr.io/enix223/alibaba-cloud-csi-driver \
    --version 1.35.1
```
